# Linage_Anjian

## 天堂M_按鍵精靈助手_腳本

### 簡介:

這是天堂M手遊的小助手,它目前的功能有:  

* 被打瞬移  
* 地監無怪瞬移
* 血量設定%數 回卷/瞬移
* 自動解副本
* 陸續增加中...

### 使用注意事項

- 必須將模擬器解析度 調成1280 * 720 

### 按鍵精靈助手

- 雙擊按鍵精靈助手無回應： 請檢查路徑，路徑不可有中文。 
- 亂碼問題： 將電腦系統地區語言全部改為大陸簡體中文
   - 設定 
   - -> 地區 -> 國家或地區 改為 中國->地區格式 改為 中文（簡體，中國）
   - -> 語言 -> 右邊 相關設定-> 系統管理語言 -> 格式 改成 中文（簡體，中國） -> 系統管理 -> 變更系統地區設定 改為中文（簡體，中國）
